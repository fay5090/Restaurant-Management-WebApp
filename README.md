# Restaurant Reservation Web App

## Project Overview
This project is a restaurant reservation web application built using HTML, CSS, and JavaScript. It allows users to book reservations and view upcoming reservations.

## Problem It Solves
The application helps restaurants manage customer reservations efficiently and allows users to easily schedule bookings.

## Technologies Used
- HTML (Semantic Structure)
- CSS (Responsive Design with Flexbox & Media Queries)
- JavaScript (DOM Manipulation & Event Handling)
- LocalStorage (Data Persistence)

## Features
- Reservation form with validation
- Data saved to localStorage
- Reservations displayed across multiple pages
- Responsive design
- User-friendly navigation

## How to Run
Open index.html in a web browser.
